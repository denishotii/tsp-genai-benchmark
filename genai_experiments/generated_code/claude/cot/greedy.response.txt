# Reasoning

1. **Problem**: Given an n×n symmetric distance matrix, build a TSP tour using nearest-neighbor heuristic starting from city 0. Visit nearest unvisited city repeatedly, then return to start.

2. **Data structures**: 
   - A boolean `visited` array of size n
   - A `tour` list to store visiting order
   - Main loop: n-1 iterations, each finds the nearest unvisited city by masking distances from current city
   
3. **Edge cases**:
   - n=1: tour = [0], length = 0
   - n=2: tour = [0, 1], length = 2 * dist[0][1]
   - Ties: `np.argmin` deterministically returns the first minimum

```python
import numpy as np

def solve(dist_matrix):
    dist_matrix = np.asarray(dist_matrix, dtype=float)
    n = dist_matrix.shape[0]
    
    if n == 0:
        return [], 0.0
    if n == 1:
        return [0], 0.0
    
    visited = np.zeros(n, dtype=bool)
    tour = [0]
    visited[0] = True
    current = 0
    
    for _ in range(n - 1):
        dists = dist_matrix[current].copy()
        dists[visited] = np.inf
        next_city = int(np.argmin(dists))
        tour.append(next_city)
        visited[next_city] = True
        current = next_city
    
    length = 0.0
    for i in range(n):
        length += float(dist_matrix[tour[i], tour[(i + 1) % n]])
    
    return tour, length
```
