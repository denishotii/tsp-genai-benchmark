# Reasoning

1. **Problem**: Given an n×n distance matrix, build a TSP tour using nearest-neighbor heuristic starting from city 0. At each step, go to the closest unvisited city. Return tour (list of indices) and total closed-tour length.

2. **Data structures**: 
   - A boolean `visited` array of size n.
   - A list `tour` to store visit order.
   - Main loop: n-1 iterations, each finding nearest unvisited city via masked argmin.

3. **Edge cases**:
   - n=1: tour=[0], length=0.
   - n=2: tour=[0,1], length = 2*d(0,1).
   - Ties: `np.argmin` picks smallest index, which is deterministic.

```python
import numpy as np

def solve(dist_matrix):
    dist_matrix = np.asarray(dist_matrix, dtype=float)
    n = dist_matrix.shape[0]
    
    if n == 0:
        return [], 0.0
    if n == 1:
        return [0], 0.0
    
    visited = np.zeros(n, dtype=bool)
    tour = [0]
    visited[0] = True
    current = 0
    length = 0.0
    
    for _ in range(n - 1):
        dists = dist_matrix[current].copy()
        dists[visited] = np.inf
        nxt = int(np.argmin(dists))
        length += float(dist_matrix[current, nxt])
        tour.append(nxt)
        visited[nxt] = True
        current = nxt
    
    length += float(dist_matrix[current, tour[0]])
    return tour, length
```
