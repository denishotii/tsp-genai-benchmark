# Reasoning

1. **Problem**: Given an n×n symmetric distance matrix, build a TSP tour using the nearest-neighbor greedy heuristic starting from city 0. At each step, go to the closest unvisited city. After visiting all, return to start.

2. **Data structures**:
   - A boolean `visited` array of size n to track visited cities.
   - A `tour` list starting with city 0.
   - Loop n-1 times: from current city, mask visited cities (set distance to infinity), pick argmin, append, mark visited.
   - Compute length by summing consecutive distances plus closing edge.

3. **Edge cases**:
   - n = 1: tour = [0], length = 0.
   - n = 2: tour = [0, 1], length = 2 * d(0,1).
   - Ties: `np.argmin` returns the smallest index, which is deterministic.

```python
import numpy as np

def solve(dist_matrix):
    dist_matrix = np.asarray(dist_matrix, dtype=float)
    n = dist_matrix.shape[0]
    
    if n == 0:
        return [], 0.0
    if n == 1:
        return [0], 0.0
    
    visited = np.zeros(n, dtype=bool)
    tour = [0]
    visited[0] = True
    current = 0
    
    for _ in range(n - 1):
        dists = dist_matrix[current].copy()
        dists[visited] = np.inf
        next_city = int(np.argmin(dists))
        tour.append(next_city)
        visited[next_city] = True
        current = next_city
    
    length = 0.0
    for i in range(n):
        length += float(dist_matrix[tour[i], tour[(i + 1) % n]])
    
    return tour, length
```
